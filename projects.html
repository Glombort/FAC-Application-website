Functions

This prompt created three functions in JavaScript getting a good idea of using strings within functions. The functions were:

  1. Greet the user

    This took in a string as input and returned Hello with the input string attached.

  2. Find the type of argument passed

    This took any argument and would return the data type of argument passed, of which 5 were possible; String, Number, Boolean, Object and Function.

  3. Find the length of a string passed through

    This would take a string as an argument and return with the length of the whole string including whitespaces. It could also take in objects such as arrays and return the number of elements in said array.

Number Functions

This prompt created 7 different functions getting a key understanding in taking number inputs with JavaScript. These functions were

  1. Summing Two Numbers

    Taking an input of 2 numbers and returning the result of them added together

  2. Add and Multiply

    Accepts an input of 4 numbers and the first two and last two, returning the product of these two sums.

  3. Return Larger

    Accepts the input of two numbers returning the larger of the two

  4. Sum of elements

    Takes an array as input returing the sum of the numbers in the array

  5. Squaring a number

    Accepts one input and returns the square of that input

  6. Odd or Even

    Accepts one input and returns whether it is 'Odd' or 'Even'

  7. Sum of Positive

    Takes an array and returns the sum of all the positive numbers in the array, if this is imposible it returns 0

DOM Manipulation

The focus of this prompt was to firstly get comfortable with the DOM (Document Object Model), but also to create a 'codename validator'.
This validator takes a string as an input and returns whether it is an acceptable codename or not. There were three criteria for the codename to be valid. It had to firstly have at least 6 characters, it also had to have at least one lowercase letter and one uppercase letter, and finally it had to have at least 2 numbers.
This prompt was a great way to understand how each part of the html, js and css work together.

Arrays and Array Methods

This prompt was similar to DOM Manipulation but with more of a focus on some neat JavaScript functionality for arrays. The prompt required using two different array methods the filter method and the reduce method.
The filter method was used to 'filter' the array only keeping multiples of two for the user.
The reduce method was used to sum all numbers in the array.
These were both really interesting to find swifter ways of using functions on arrays. It was also a great way to think about what a user could enter and how to get around parts of what a user inputs that are unnecessary or incorrect.
